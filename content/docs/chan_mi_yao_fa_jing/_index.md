# 禅秘要法经


亦称《禅经秘要法》、《禅秘要法》。佛教经典。后秦鸠摩罗什编译。三卷。是鸠摩罗什受弟子僧睿之请,综合印度大小乘禅法编译而成。全书倡说五门禅观:不净观、慈悲观、因缘观、数息观、念佛观。
